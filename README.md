# DSA Problem Tracker

A clean, minimal web app to track your daily DSA grind. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- Log problems with name, topic, difficulty, and notes
- Filter by Easy / Medium / Hard
- Search by name or topic
- Track your solve streak and counts
- All data saved locally in your browser (localStorage)

## How to use

1. Clone the repo
2. Open `index.html` in your browser
3. Start logging problems

```bash
git clone https://github.com/YOUR_USERNAME/dsa-tracker.git
cd dsa-tracker
open index.html
```

## Tech stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript
- localStorage for persistence
